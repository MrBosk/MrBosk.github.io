Hello World

2017-07-27

♠ 前言

欢迎来到Mosk的博客天地！

搭建博客的想法来自于偶然在简书上看到的[利用 GitHub Pages 快速搭建个人博客—BYQiu]。

然而，实际上我对照搭建起来后发现，由于代码是Fork过来的缘故，我不知道怎样改成自己喜欢的风格。又因为当时比较忙，遂，暂时放弃了搭建博客的想法。

最近，因为学到了新的东西，遂想写一篇博客压压惊。

刚好最近闲了下来，利用早上起来和晚上下班后的时间，找资料查代码，搭建了一个我的博客天地。

废话不多说，直接看正文。

♠ 正文

接下来说说博客搭建的技术应用。

搭建博客是以发布到GitHub的GitHubPage上为目的。

以此为搜索依据，博主搜到以下几种方案：

  • GitHub Pages + Jekyll
  • GitHub Pages + Node.js + Hexo
  • GitHub Pages + Node.js + Express

博主选择的是第二种方案，原因嘛，有以下几条：

  1. Hexo + Node.js + GitHub这个方案是搜索量最多的。
  2. 对Node.js这一前端运行环境的好奇。
  3. Express是基于Node.js平台的Web开发框架，并不是专门用来搭建博客的，获取它在Web开发方面可能更好，但是我在搭建博客方面优选Hexo。

主题来自PPOffice的主题[Hueman]。

从[Hueman]Fork过来，下载并按照主题作者的配置方案进行配置。

由于对页面的部分不是很满意，遂，自己对主题的字体、样式等方面做了些小的改动，以适应自己的风格。

♠ 后记

最后，感谢众多大佬提供的搭建博客资料，感谢PPOffice提供的主题[Hueman主题]。

在下一篇博客中，我也会将搭建博客的详细过程记录下来，供大家参考。

希望你也能喜欢我的博客风格，如果感兴趣的话，可以参照个人GitHub博客搭建指南，搭建出你的博客天地。
